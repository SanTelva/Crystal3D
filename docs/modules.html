Модули

Результаты работы мы сформировали в один большой модуль draw2. В него же просим вас складывать то, что написали вы. Оглавление:

  • Модуль draw2.evaluations содержит различные вычисления, связанные с точками.
  • Модуль draw2.maths содержит матрицы, векторы, и функции для работы с ними.
  • Модуль draw2.graphics содержит в себе функции, предназначенные для отрисовки кристаллов на экране
  • Модуль draw2.nbolib содержит классы системы NBO, а также функции для работы с ними.
  • Модуль draw2.nboio содержит функции для ввода/вывода файлов в формате NBO.
  • Модуль draw2.oldio содержит функции для ввода/вывода файлов NBO, совместимые с старым кодом, использующим primitiveCell.
  • Модуль draw2._legacy содержит старые функции, которые использовать не надо.
Конкретную документацию можно посмотреть из меню слева (оно слева).
