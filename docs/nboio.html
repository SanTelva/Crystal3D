Модуль draw2.nboio

Модуль для ввода и вывода данных в формате NBO. Первое, что следует знать об этом модуле — в нём есть функция nboOutput() с суффиксом, т.е. кончающиеся на 2D или 3D. Если вы знаете, с каким типом данных работаете — с двумерным или трёхмерным — лучше используйте их. Если не знаете — вызовите функцию без суффикса и кидайте d20 на удачу. Это небольшой модуль — в нём всего две функции (без учёта суффиксов):
nboInput(filename) — читает из файла NBO. Возвращает angle, a, b, nodes, bonds для двумерного файла или angleA, angleB, angleG, a, b, c, nodes, bonds для трёхмерного. angle — угол двумерной элементарной ячейки, angleA, angleB и angleG — углы трёхмерной. a, b и c — длины рёбер элементарной ячейки.
nboOutput(filename, angle, a, b, nodes, bonds) — двумерная функция. Лучше использовать с суффиксом 2D. Выводит данные в файл filename. Перед использованием убедитесь, что nodes и bonds синхронизированы (т.е. в них одинаковые данные).
nboOutput(filename, angle, a, b, c, nodes, bonds) — трёхмерная функция. Лучше использовать с суффиксом 3D. Выводит данные в файл filename. Тут есть одна тонкость — angle это tuple из трёх элементов — angleA, angleB и angleG.
