Модуль draw2.oldio

Модуль, написанный для обратной совместимости. Реализует функции чтения и записи в файл NBO, с точки зрения пользователя полностью аналогичные старым функциям чтения и записи в файл IGOR. Вот эти функции:
myInput(filename) -> primitiveCell — читает данные из файла filename. Возвращает IGOR's primitiveCell, содержащую и топологическую, и геометрическую информацию (подробнее в разделе IGOR и IGOR3D).
myOutput(filename, primitiveCell) -> None — пишет данные в файл filename. primitiveCell — это IGOR's primitiveCell, содержащая как топологическую, так и геометрическую информацию.
